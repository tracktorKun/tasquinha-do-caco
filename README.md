
Tasquinha do Caco - Website
Este projeto consiste numa proposta de reformulação para o website do restaurante Tasquinha do Caco, focado na modernização da interface e na experiência do utilizador.

Tecnologias
React 19: Biblioteca para a construção da interface.

Vite: Ferramenta de build e servidor de desenvolvimento.

Tailwind CSS v4: Framework para estilização e design responsivo.

Google Gemini AI: Integração com o modelo Gemini via @google/genai.

Motion: Biblioteca para a implementação de animações.

Instalação e Execução
Instalar dependências:

Bash
npm install
Configuração:

Copie o ficheiro .env.example para um novo ficheiro chamado .env.local.

Insira a sua GEMINI_API_KEY no ficheiro criado.

Executar o projeto:

Bash
npm run dev
